Brief
================
在setting.py里面填写帐号密码后,可获取当前学期的西科大课程表  
运行本脚本，需要安装BeautifulSoup模块，我安装了[bs4版本](http://www.crummy.com/software/BeautifulSoup/bs4/download/4.1/)
--
Future
================
items.py定义了一些功能项，目前只完成了课程表部分
